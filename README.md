# d88li_sidequest_9

---

## Description

A dungeon platformer where the player, a thief, attempts to steal all the apples from the patrolling suspicious merchants. Player can jump, attack, and run. Damage is incurred when the player touches the fire, or makes contact with a merchant. Each time damage is incurred the player loses one heart out of their total 3. Once all their hearts are out, the player loses.

---

## Set up and interaction instructions

Up arrow key: Jump
Left arrow key: Run left
Right arrow key: Run right
Space bar: Attack

Navigate the platforms by jumping and running, attack the merchants as necessary. Collect as many apples as possible as you navigate through the map.

---

## Debugging

`: toggle debug menu on and off
Space bar: turning a setting on and off

Developer can show enemy probes, show collision boxes, make player invincible, and set the win condition to one.
